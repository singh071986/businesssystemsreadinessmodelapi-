Business Systems Readiness Assessment

Project Overview

This web application delivers a personalized business pathway and PDF report for users, styled to match Easy Artisan Websites branding and integrated with your business API.

Key Features

  • Modern, responsive UI (Bootstrap + custom branding)
  • Interactive assessment form
  • API integration for real-time results
  • PDF report download with custom filename
  • Custom animated loading spinner
  • Clear/reset answers and report

Technology Stack

  • Python 3, Flask (backend)
  • HTML, CSS (Bootstrap), JavaScript (frontend)
  • Jinja2 templating
  • xhtml2pdf for PDF generation
  • requests for API calls
  • pytz for timezone handling

Setup & Deployment

  1. Clone the repository and navigate to flask_ui
  2. Create and activate the virtual environment
  3. Install dependencies from requirements.txt
  4. Run python app.py and open http://localhost:5000

Maintenance & Management

  • Update dependencies via requirements.txt
  • Update branding by replacing static assets and editing index.html
  • API changes: update API_URL and payload in app.py
  • Edit pdf_report.html for PDF layout changes
  • Replace spinner_2.gif for spinner updates
